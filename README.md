## Installation

 - Get it locally -
   ```sh
   $ git clone https://github.com/psikarwal/contacts.git
   ```

   **OR**

   Download the latest version here: [Contacts](https://codeload.github.com/psikarwal/contacts/zip/master)

 - Open terminal.
 - `cd medium`.
 - `npm install`.
 - `npm start`
